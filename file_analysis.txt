PROJECT OVERVIEW:
This is an email filtering and processing system that integrates with Microsoft Outlook using Windows COM automation. The system extracts emails from Outlook, applies custom filters, and processes them based on Power Automate-like expressions.

CORE FILES ANALYSIS:

1. test.py (MAIN CORE FILE - 396 lines)
   PURPOSE: Primary email processing engine with Outlook integration
   FUNCTIONALITY:
   - EmailFilterManager class for managing email filters
   - Power Automate expression language evaluation
   - Outlook COM integration to read emails from inbox
   - Filter application and email processing
   - JSON export of filtered emails
   KEY FEATURES:
   - Supports complex filter expressions (equals, contains, startsWith, etc.)
   - Email filtering based on sender, subject, body content
   - Action-based email processing
   - Integration with iSupport email system
   STATUS: ESSENTIAL - Core system functionality

FILTER SYSTEM:
The system uses Power Automate-like expressions for email filtering:
- equals(), contains(), startsWith(), endsWith()
- concat(), substring(), indexOf(), split()
- Logical operators: and(), or(), if()
- Email fields: sender, subject, body, date, recipients
